# 디지털 게시판 시스템 (MVP)

교회 입구/식당 등 다중 사이트에 설치된 디스플레이를 중앙에서 통합 제어하는 디지털 사이니지 시스템입니다.

## 시스템 구성

```
┌─────────────────────────────────────────────────────────┐
│  호스트 PC (관리자)                                      │
│  ┌───────────────────────────────────────────────────┐  │
│  │  호스트 서버 (Node.js + Express + WebSocket)       │  │
│  │  - 웹 관리 UI (http://localhost:3000)             │  │
│  │  - 콘텐츠 업로드/관리                              │  │
│  │  - 클라이언트 인식 및 재생 명령 전송               │  │
│  └───────────────────────────────────────────────────┘  │
└─────────────────────────────────────────────────────────┘
         │ WebSocket (LAN)
         ├──────────────────────────────────┐
         ▼                                  ▼
┌─────────────────────┐      ┌─────────────────────────┐
│  현관 PC (모니터 1대) │      │  식당 PC (모니터 2대)    │
│  클라이언트 플레이어  │      │  클라이언트 플레이어      │
│  (Electron)          │      │  (Electron)              │
└─────────────────────┘      └─────────────────────────┘
```

## 빠른 시작

### 사전 요구사항

- **Node.js 18 이상** (https://nodejs.org)
- 호스트 PC와 클라이언트 PC가 **같은 네트워크(LAN)**에 있어야 합니다.

---

### 1. 호스트 서버 설치 및 실행

```bash
cd host
npm install
npm start
```

서버가 시작되면 브라우저에서 `http://localhost:3000` 으로 관리 화면에 접속합니다.

---

### 2. 클라이언트 플레이어 설치 및 실행

```bash
cd client
npm install
```

**호스트 서버 주소 설정** (최초 1회):

`client/config.js` 파일을 열어 호스트 PC의 IP 주소로 변경합니다:

```javascript
module.exports = {
  HOST_URL: 'ws://192.168.0.10:3000',  // ← 호스트 PC의 실제 IP
  CLIENT_NAME: '현관'                   // ← 이 PC의 식별 이름
};
```

실행:
```bash
npm start
```

> **참고**: 클라이언트는 전체화면으로 실행됩니다. `ESC` 키로 종료할 수 있습니다.

---

### 3. 사용 방법

1. 호스트 관리 화면(`http://localhost:3000`)에서 **연결된 클라이언트** 확인
2. **콘텐츠 업로드** (이미지/동영상 드래그 앤 드롭)
3. 재생할 **클라이언트 선택** (클릭)
4. 재생할 **콘텐츠 선택** (1~2개 클릭)
5. **▶ 재생** 버튼 클릭

---

## 모니터 개수별 동작

| 클라이언트 모니터 | 콘텐츠 1개 선택 | 콘텐츠 2개 선택 |
|---|---|---|
| 1대 | 전체화면 재생 | 순차 재생 (슬라이드쇼) |
| 2대 | 좌측 모니터 재생 | 좌/우 모니터 각각 재생 |

---

## 부팅 시 자동 실행 설정 (Windows)

1. `Win + R` → `shell:startup` 입력
2. 해당 폴더에 바로가기 생성:
   - 대상: `"C:\Program Files\nodejs\node.exe" "C:\signage\client\node_modules\.bin\electron" "C:\signage\client"`
   - 또는 빌드된 `.exe` 파일의 바로가기

## 프로젝트 구조

```
digital-signage/
├── host/                    # 호스트 서버
│   ├── server.js           # 메인 서버 (Express + WebSocket)
│   ├── public/
│   │   └── index.html      # 관리자 웹 UI
│   ├── uploads/            # 업로드된 콘텐츠 저장소
│   └── package.json
├── client/                  # 클라이언트 플레이어
│   ├── main.js             # Electron 메인 프로세스
│   ├── preload.js          # IPC 브릿지
│   ├── player.html         # 렌더러 (재생 화면)
│   ├── config.js           # 설정 (호스트 URL, 이름)
│   └── package.json
└── README.md
```
