  디지털 게시판 클라이언트 — 설치 안내
════════════════════════════════════════════════════════

■ 준비물
  - Windows PC
  - Node.js LTS (없으면 install.bat 실행 시 다운로드 페이지가 열립니다)
    → https://nodejs.org 에서 LTS 버전 설치

■ 설치 (한 번만)
  1) [중요] 압축 풀기 전에 zip 차단 해제
     - 다운로드한 signage-client.zip 오른쪽 클릭 → 속성
     - 창 하단의 "차단 해제(Unblock)" 체크 → 확인
     - (이 단계를 건너뛰면 install.bat 이 보안 정책으로 차단될 수 있습니다)
  2) zip 을 원하는 폴더에 압축 해제 (예: C:\signage-client)
  3) install.bat 더블클릭
     - 필요한 프로그램 자동 설치
     - 윈도우 부팅 시 자동 실행 등록
     - 설치가 끝나면 클라이언트가 자동 실행됩니다

■ "애플리케이션 제어 정책에서 차단" / "웹의 위험한 파일" 오류가 뜰 때 (Windows 11)
  아래를 순서대로 시도하세요.

  [방법 1] 파일 차단 해제
   - install.bat (또는 start.bat) 오른쪽 클릭 → 속성
     → 하단 "차단 해제" 체크 → 확인 → 다시 실행
   - 여러 파일을 한 번에: PowerShell(관리자)에서
       Get-ChildItem 'C:\signage-client' -Recurse | Unblock-File

  [방법 2] 그래도 "애플리케이션 제어 정책" 메시지가 나오면
           = 스마트 앱 제어(Smart App Control)가 켜져 있는 것입니다.
   - 설정 → 개인정보 및 보안 → Windows 보안
     → 앱 및 브라우저 컨트롤 → "스마트 앱 제어" → 끄기
   - ※ 이 설정은 한 번 끄면 다시 켤 때 Windows 재설치가 필요합니다.
     사이니지 전용 PC 에서만 끄세요.

  [방법 3] 위가 어려우면 install.bat 없이 수동 설치
   - 폴더에서 주소창에 cmd 입력 후 Enter → 검은 창에서 아래 순서로:
       npm install --production
       npx electron .
   - (부팅 시 자동 실행은 start.bat 을 시작프로그램 폴더에 넣으면 됩니다.
      Win+R → shell:startup → start.bat 바로가기 복사)

■ 최초 설정
  1) 실행되면 설정 화면이 나타납니다
  2) 클라이언트 이름 입력 (예: 현관, 식당)
  3) 호스트 서버 주소 입력: https://signage.yebom.org
  4) 모니터 수 선택 (1 또는 2)
  5) "연결 및 시작" 클릭
  6) 관리자가 승인하면 자동으로 전체화면 재생이 시작됩니다

■ 평소 실행
  - PC 를 켜면 자동으로 실행됩니다
  - 수동 실행: start.bat 더블클릭

■ 새 버전 업데이트
  이 zip 은 "신규 설치 · 업데이트 겸용" 입니다. 새 zip 을 받으면:
  1) 앱 종료: Ctrl+Shift+Q  (실행 중이면 새 창이 겹쳐 뜰 수 있음)
  2) 새 zip 을 "기존 설치 폴더에 그대로" 덮어쓰기 압축 해제
     (같은 폴더여야 함 — node_modules 가 유지됨)
  3) install.bat 실행  (신규/업데이트를 자동 판별합니다)
     → 더 빠르게 하려면 install.bat 대신 start.bat 만 실행해도 됩니다.
  ※ config.json(설정)·cache(편성표/미디어)는 zip 에 없으므로 항상 보존됩니다.
  ※ 새 PC 든 기존 PC 든 "압축 풀고 install.bat" 절차가 똑같습니다.

■ 화면 조작 (마우스)
  - 화면에서 마우스를 움직이면 하단에 조작 바가 나타납니다:
    [⏹ 멈추기 / ▶ 재생하기]  [⚙ 설정]  [⏻ 앱 종료]
  - 멈추기: 재생만 멈춤(화면에 "재생 종료" 표시). 재생하기로 다시 시작.
  - 앱 종료: 프로그램 자체를 끔(부팅 시 자동 재실행됨).
  - 호스트 관리 화면에서도 멈추기/재생하기가 가능합니다.
  - 3초간 움직임이 없으면 조작 바가 자동으로 사라집니다.

■ 단축키
  Ctrl+Shift+P : 멈추기 / 재생하기
  Ctrl+Shift+Q : 앱 종료 (확인창 표시)
  Ctrl+Shift+S : 설정 화면 다시 열기
  Ctrl+Shift+F : 전체화면 켜기/끄기
  ESC          : 전체화면 해제

■ 자동시작 해제
  - uninstall.bat 실행

■ 특징
  - 호스트가 꺼져 있거나 인터넷이 끊겨도, 마지막으로 받은
    편성표와 캐시된 미디어로 계속 재생됩니다.
