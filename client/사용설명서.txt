  디지털 게시판 클라이언트 — 설치 안내
════════════════════════════════════════════════════════

■ 준비물
  - Windows PC
  - Node.js LTS (없으면 install.bat 실행 시 다운로드 페이지가 열립니다)
    → https://nodejs.org 에서 LTS 버전 설치

■ 설치 (한 번만)
  1) 이 zip 을 원하는 폴더에 압축 해제 (예: C:\signage-client)
  2) install.bat 더블클릭
     - 필요한 프로그램 자동 설치
     - 윈도우 부팅 시 자동 실행 등록
     - 설치가 끝나면 클라이언트가 자동 실행됩니다

■ 최초 설정
  1) 실행되면 설정 화면이 나타납니다
  2) 클라이언트 이름 입력 (예: 현관, 식당)
  3) 호스트 서버 주소 입력: https://signage.yebom.org
  4) 모니터 수 선택 (1 또는 2)
  5) "연결 및 시작" 클릭
  6) 관리자가 승인하면 자동으로 전체화면 재생이 시작됩니다

■ 평소 실행
  - PC 를 켜면 자동으로 실행됩니다
  - 수동 실행: start.bat 더블클릭

■ 화면 조작 (마우스)
  - 화면에서 마우스를 움직이면 하단에 조작 바가 나타납니다:
    [⏸ 일시정지 / ▶ 재개]  [⚙ 설정]  [⏻ 종료]
  - 3초간 움직임이 없으면 자동으로 사라집니다.

■ 단축키
  Ctrl+Shift+P : 일시정지 / 재개
  Ctrl+Shift+Q : 종료 (확인창 표시)
  Ctrl+Shift+S : 설정 화면 다시 열기
  Ctrl+Shift+F : 전체화면 켜기/끄기
  ESC          : 전체화면 해제

■ 자동시작 해제
  - uninstall.bat 실행

■ 특징
  - 호스트가 꺼져 있거나 인터넷이 끊겨도, 마지막으로 받은
    편성표와 캐시된 미디어로 계속 재생됩니다.
