클라이언트 설치

💻 PC 전용 클라이언트 (Electron)

전체화면 키오스크 + 오프라인 캐시 재생. 압축 해제 후 install.bat 실행 한 번으로 설치·부팅 자동실행까지 완료됩니다.

⬇ 설치 파일 다운로드 (zip)

⚠ Windows 11에서 install.bat이 "애플리케이션 제어 정책"으로 차단되면: 압축 풀기 전 zip 오른쪽클릭 → 속성 → 차단 해제. 그래도 막히면 설정 → Windows 보안 → 앱 및 브라우저 컨트롤 → 스마트 앱 제어 끄기 (동봉 사용설명서 참고).

🌐 웹 플레이어

브라우저에서 바로 재생. 설치 없이 테스트하거나 간단히 띄울 때 사용합니다.

초기화 (새출발)

클라이언트 등록 초기화

등록·승인된 모든 클라이언트를 지웁니다. 연결된 클라이언트는 대기 화면으로 돌아가 재승인이 필요합니다. (사이트·편성표는 유지)

전체 초기화

클라이언트 + 사이트 + 편성표를 모두 비웁니다. 완전히 새로 시작할 때만 사용하세요. (콘텐츠 파일은 유지)
